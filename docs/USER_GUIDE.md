# Resconate User Guide

## Getting Started

### For HR Managers

1. **Login**
   - Navigate to `/hr-login`
   - Enter your HR credentials
   - Click "Login"

2. **Dashboard Overview**
   - View employee statistics
   - See recent activities
   - Access quick actions

3. **Employee Management**
   - View all employees
   - Add new employees
   - Edit employee information
   - Export employee data

4. **Payroll Management**
   - Process payroll
   - View payroll history
   - Generate payslips
   - Export payroll reports

5. **Leave Management**
   - View leave requests
   - Approve/reject requests
   - Track leave balances

6. **Recruitment**
   - Post job openings
   - Manage candidates
   - Schedule interviews
   - Track application status

### For Employees

1. **Login**
   - Navigate to `/employee-login`
   - Enter your employee credentials

2. **Employee Portal**
   - View your profile
   - Access payslips
   - Request leave
   - View performance reviews
   - Update personal information

### For Administrators

1. **Login**
   - Navigate to `/admin-dashboard`
   - Use admin credentials

2. **Admin Dashboard**
   - System overview
   - User management
   - Onboarding tasks
   - Jobs management
   - Candidates management
   - Audit logs
   - System settings

## Features

### Compliance Calculators

1. Navigate to `/compliance-calculators`
2. Select calculator type (PAYE, NSITF, ITF, PenCom, NHF)
3. Enter gross salary
4. Select state (for PAYE)
5. View calculation results
6. Export reports if needed

### Banking Integration

1. Navigate to `/banking`
2. Connect bank accounts
3. Verify account numbers
4. View payment transactions
5. Process bulk payments
6. Generate reconciliation reports

### Payment & Billing

1. Navigate to `/payment`
2. Select payment provider
3. Choose payment method
4. View subscription status
5. Manage invoices
6. Update payment methods

### Analytics

1. Navigate to `/analytics`
2. View key metrics
3. Analyze trends
4. Compare benchmarks
5. Export reports

### Referrals

1. Navigate to `/referrals`
2. Copy referral link
3. Share with friends
4. Track referrals
5. View rewards

## Keyboard Shortcuts

- `Ctrl/Cmd + K` - Quick search (if implemented)
